## 切比雪夫不等式

### 定义

$$
P(| X - \Epsilon(X) | \geq \epsilon) <  \frac{\sigma^2}{n \epsilon^2}
$$

| 参数 | 说明     |
| ---- | -------- |
| X    | 随机变量 |
| $$\Epsilon(X)$$     | X的数学期望         |
|  n    |     实验次数(样本数)     |
| $$\epsilon$$     |   误差       |
|  $$\sigma^2$$    |   方差       |

### 意义

当观察到的数据足够多时，随机性和噪声的影响可以忽略不计
